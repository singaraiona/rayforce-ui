Zero-Copy

Pointer exchange between threads. No serialization, no marshaling — raw column buffers rendered directly.

;; Create a grid widget
(set grid1 (widget {type: 'grid name: "trades"}))

;; Push live query results
(draw grid1 (select {from: trades where: (> price 100)}))

;; Callback on row select
(set grid1 (widget {type: 'grid
  on-select: (fn [row] (draw details row))}))

;; Timer-driven live dashboard
(timer 100 10000000 (fn [x]
  (draw t (take trades -100))
  (draw pc (select {from: (take trades -500)
    by: (xbar Ts 1000) open: (first Price)
    high: (max Price) low: (min Price)
    close: (last Price)}))))

Widgets are Rayfall expressions

Every widget is a first-class Rayforce object. Create grids, charts, and text displays with widget, push data with draw, and attach callbacks — all in a single expression language.

Queries execute in the Rayforce worker thread and pass raw column pointers to the UI. No copies, no marshaling — just pointer exchange.
